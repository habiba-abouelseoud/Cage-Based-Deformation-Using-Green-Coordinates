Load the cage from a file. 

This function loads the cage from a file and populates the cage\+\_\+cell\+\_\+ and cage\+\_\+nods\+\_\+ matrices. It reads the number of cage elements and vertices, followed by the vertex coordinates. The cage\+\_\+cell\+\_\+ matrix is resized to have 3 rows and the number of cage elements columns, and the cage\+\_\+nods\+\_\+ matrix is resized to have 3 rows and the number of cage nodes columns. The coordinates of the cage elements and nodes are stored in the respective matrices. The rest segment element lengths are calculated and stored in the cage\+\_\+rest\+\_\+uv\+\_\+ matrix. The initial outward normal vectors of the cage elements are calculated and stored in the cage\+\_\+normal\+\_\+ matrix.

Load the model points from a file.
